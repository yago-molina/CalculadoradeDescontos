# 📐 Calculadora de Descontos

## 🎯 Objetivo do Projeto
Este repositório contém um projeto prático desenvolvido com foco em duas áreas fundamentais do desenvolvimento web: **Design de Interfaces** e **Precisão Matemática**. 

A atividade foi construída passo a passo sob a orientação do professor, com o intuito de aplicar conceitos teóricos na prática. O grande desafio e aprendizado deste projeto foi unir uma experiência visual agradável e intuitiva para o usuário (front-end) com cálculos lógicos e exatos rodando nos bastidores.

## 🚀 Tecnologias Utilizadas
* **HTML5:** Estruturação semântica do conteúdo.
* **CSS3:** Estilização, layout e princípios de UI/UX para uma interface moderna.
* **JavaScript:** Lógica de programação, manipulação do DOM e garantia de cálculos matemáticos precisos.

## 🧠 Focos de Aprendizagem
* **Design de Interfaces:** Construção de layouts responsivos, uso adequado de espaçamentos, tipografia e cores para guiar a atenção do usuário.
* **Precisão Matemática:** Implementação de algoritmos no JavaScript para processar dados de forma exata, lidando com operações matemáticas, tratamento de variáveis e atualização em tempo real da interface com base nos resultados.

## 📚 Contexto
Este projeto faz parte das atividades práticas de aprendizagem do curso de **Desenvolvimento de Sistemas** do **Senai Cruzeiro**, visando o aprimoramento em desenvolvimento front-end e resolução de problemas lógicos.
